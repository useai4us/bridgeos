## 为什么是 BridgeOS？

在多个 AI 平台之间来回切换时，你常常会遇到：

- 想把一段上下文发到另一个平台，复制粘贴很麻烦
- 好用的提示词和对话片段越积越多，却很难复用
- 换设备后，历史沉淀和设置不同步

**BridgeOS** 就是为这个场景设计的：  
把「迁移 → 保存 → 同步」做成一条顺滑工作流。

---

## 你可以用 BridgeOS 做什么？

- **跨平台一键发送**  
  在任意网页选中文本，快速发送到 ChatGPT / Claude / Gemini / NotebookLM。
- **本地对话库（Library）**  
  把有价值的对话沉淀下来，支持搜索、筛选、再次投递。
- **输入框 `$` 快速引用**  
  在 ChatGPT / Claude / Gemini 输入框按 `$`，直接检索并插入历史内容。
- **云同步（可选）**  
  登录后跨设备同步 Library、设置与导出缓存。
- **会员能力（Pro）**  
  提供更高配额与更多在线能力（支持月/季/年方案）。

## 3 步开始使用

1. **安装扩展**并固定到浏览器工具栏  
2. **（可选）开启 Cloud Sync**，用 Google 登录  
3. 在常用 AI 平台上开始迁移、保存、引用你的对话内容

## 适合谁？

- 需要在多个 AI 平台协作的人
- 有大量提示词/对话资产，需要长期沉淀的人
- 追求高效率、希望减少复制粘贴的人
- 希望在多设备保持一致工作流的人

## 隐私与控制

BridgeOS 采用**本地优先**思路：  
你可以在不登录的情况下使用核心本地能力。  
云同步与会员能力仅在你主动开启后生效。
